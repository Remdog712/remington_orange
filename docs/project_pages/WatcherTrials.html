Watcher Trials

Watcher Trials is a solo VR game I developed as the final project for Game 160 - Video Game Cultures. In this project, I created all 3D assets using Blender and Substance Painter, and designed 2D assets in Illustrator. The game features a series of trials where players, scaled down to the size of two quarters, navigate challenges inspired by the movie Honey, I Shrunk the Kids. Set in a lore-driven world where a shadowy corporation shrinks captives for secret cognitive experiments, players face four trials: a short maze, an object hunt, a shooting gallery, and a logic puzzle. Upon completing the trials, they can choose to escape through an unmarked tunnel or endure more challenges as directed by the overseer. Developed over several weeks in approximately 20 hours, this project allowed me to learn more about Unreal Engine's VR Development system as well learn more about modeling and texturing for optimization of VR gameplay.
