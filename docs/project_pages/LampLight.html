Lamplight

Lamplight is a game I made alongside another student in Unreal Engine 5 as part of a final project for Game 250 - Technical Game Design in which we were tasked with a making a short game that focused on one main gameplay feature that was functioning and bug free! Functioning? Yes! Bug free? Debatable! The game took roughly 3 months from start to finish. The has been built and is able to be run standalone, though is not available through this site.
