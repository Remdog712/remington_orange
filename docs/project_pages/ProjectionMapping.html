Projection Mapping Systems

This project includes two projection systems I set up for interactive and immersive displays. The first uses a projector aimed at a set of four whiteboards, where content is projection-mapped to a defined outline. This setup is used to display information or construct “virtual window” scenes by precisely aligning imagery to the board layout.

The second system is a ceiling-mounted Kinect V2 and projector aimed down at the floor. The Kinect captures depth maps, which are then aligned to the projected image using a homography calibration. This calibration establishes a reliable correspondence between the Kinect’s depth data and the physical floor plane, allowing the system to accurately detect floor position and user interaction.

With this calibration in place, the projector overlays interactive visuals or games directly onto the ground, enabling responsive experiences where projected content reacts in real time to movement within the space.
